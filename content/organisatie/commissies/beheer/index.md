Het beheerteam draagt zorg voor het up en running blijven van NLUUG systemen en het updaten en beveiligen ervan. Daarnaast beheren zij ook de DNS en mailomgeving van NLUUG.

Het beheerteam beheert onder andere:
* De ftp-server <https://ftp.nluug.nl>
* De big blue button server <https://meet.nluug.nl>
* De Forgejo git repository server <https://git.nluug.nl>
* De Wiki server <https://wiki.nluug.nl>
* De webserver <https://www.nluug.nl>
* De Postorius + hyperkitty + mailman mailinglistserver <https://lists.nluug.nl>
* De PretalX CFP server <https://cfp.nluug.nl>
* De VPN server <https://vpn.nluug.nl>
* De LDAP server <https://ldap.nluug.nl>
* De Mastodon server <https://nluug.social>
* De Peertube server <https://video.nluug.nl>

De meeste van deze servers draaien Debian Linux en worden beheerd op basis van Ansible.
